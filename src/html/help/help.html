R Help Documentation

This is the Krarkode Help Panel. To view documentation for R functions, use the following methods:

  • Place cursor on a function name and press F1
  • Right-click and select Look Up Help
  • Use the command palette: Help: Look Up Help at Cursor

Keyboard Shortcuts

Action Windows/Linux macOS
Open Help Panel Ctrl+Shift+H Cmd+Shift+H
Look Up Help at Cursor F1 F1
Find in Help Ctrl+F Cmd+F

Getting Started

  1. Start an R Session: Use the command Ark: Open Console to start a new R session
  2. Attach to Existing Session: Use Ark: Attach to Session to connect to a running R process
  3. Write R Code: Use the editor to write R code with full LSP support (completion, hover, diagnostics)
  4. View Plots: Use the Plots panel to view and navigate plot history
  5. Browse Help: Use this Help panel to explore R documentation

Features

Language Server (LSP)

Full R language support including:

  • Code completion
  • Hover documentation
  • Diagnostics and linting
  • Go to definition
  • Find references

Plot Viewer

Interactive plot viewing with:

  • Plot history navigation
  • Zoom controls
  • Save to PNG/SVG
  • Dynamic resizing

Help System

Integrated R help documentation with:

  • Quick help lookup at cursor
  • Full help panel browser
  • Navigation history
  • Theme-aware styling

Commands

Command Description
Ark: Open Console Start a new Ark R session
Ark: Attach to Session Attach to an existing R session
Ark: Stop Session Stop the current R session
Ark: Restart Language Server Restart the R LSP server
Help: Open Help Panel Open the Help panel
Help: Look Up Help at Cursor Show help for the function at cursor
